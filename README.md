# Explore India Tours

A responsive travel website developed using HTML5, CSS3, and JavaScript.

## Features
- Responsive design for mobile, tablet, and desktop
- SEO-friendly pages
- Travel destinations and tour packages
- Interactive gallery with filters
- Package cost calculator
- Booking form validation
- Mobile navigation menu

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub
- Netlify
